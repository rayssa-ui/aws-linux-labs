# AWS Linux Labs 🐧☁️

Repositório com os aprendizados dos laboratórios práticos da AWS Academy.

## O que aprendi

### Linux
- Navegação de pastas (cd, ls, pwd)
- Criação e edição de arquivos (touch, vim, nano)
- Permissões de arquivos (chmod)
- Compactação de arquivos (tar)
- Agendamento de tarefas (cron)
- Análise de processos (ps, top)
- Comandos de texto (grep, sort, cut, sed, tee)
- Conexão SSH via PuTTY

### Python
- Script para encontrar números primos entre 1 e 250
- Salvamento de resultados em arquivo .txt

### AWS
- Criação e conexão com instâncias EC2
- Gerenciamento de usuários no Linux
- Configuração de variáveis de ambiente (PATH)

## Scripts
- `primes.py` - Encontra todos os números primos entre 1 e 250
- `results.txt` - Resultado dos números primos gerados
